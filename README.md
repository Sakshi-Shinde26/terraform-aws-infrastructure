Terraform AWS Infrastructure

This project demonstrates Infrastructure as Code (IaC) using Terraform on AWS.

Technologies Used

- Terraform
- AWS EC2
- AWS VPC
- Security Groups
- IAM

Project Overview

This project provisions AWS infrastructure using Terraform, including:

- Virtual Private Cloud (VPC)
- EC2 Instance
- Security Groups
- IAM Configuration

Benefits

- Infrastructure Automation
- Consistent Deployments
- Version Controlled Infrastructure
- Reduced Manual Configuration

Commands

terraform init

terraform plan

terraform apply

terraform destroy
